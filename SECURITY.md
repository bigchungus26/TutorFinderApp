# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Teachme, please report it responsibly.

**Email:** security@teachme.app

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This policy covers the Teachme web application and its associated APIs.

## Out of Scope

- Social engineering attacks
- Denial of service attacks
- Issues in third-party dependencies (report these to the upstream maintainer)

## Supported Versions

Only the latest version deployed to production is supported with security updates.
